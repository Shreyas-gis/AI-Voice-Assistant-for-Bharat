# Project Title: AI Voice Assistant for Rural Bharat
## Problem Statement
Rural citizens struggle to access government schemes due to lack of digital literacy and language barriers.

## Solution
An AI-powered multilingual voice assistant that explains government schemes in regional languages.

## Features
- Voice input in Kannada/Hindi
- Government scheme database
- AI-based query understanding
- Works in low internet mode
- Mobile compatible

## Target Users
Rural citizens, senior citizens, farmers

## Impact
Improves accessibility to welfare schemes.
