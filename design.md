# System Design
## Architecture
User → Mobile App → Speech to Text → NLP Model → Govt Database → Response
## Technologies Used
- Python
- AWS
- OpenAI / LLM
- Speech Recognition API
- Mobile frontend (Flutter)

## Data Flow
1. User speaks query
2. Converted to text
3. NLP model understands intent
4. Fetch relevant scheme
5. Convert answer to speech

## Future Scope
- Add more regional languages
- Offline support
- Integration with CSC centers
